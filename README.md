
<h1>PySpark Basics – DataFrame Practice</h1>

This project demonstrates basic operations in PySpark DataFrames using simple customer data.
It helps in understanding how to perform common data queries such as filtering rows, selecting columns, and counting records.

The goal of this practice is to become comfortable with the basic syntax of PySpark and understand how data can be processed step by step.

<h2>Dataset Used</h2>
<b>Customers Data</b>

Contains basic details of customers like name, city, and age.

<b>Orders Data</b>

Contains order information such as order amount and date.
